# WCA 查询 —— Astrbot 插件

WCA 与 one 成绩查询插件。采用全 API 架构，轻量化设计，不拉取数据库，实时查询 WCA 官方数据与 one 平台数据。

**当前版本：v1.1.6**

## 功能

- **成绩查询**：查询 WCA 官方成绩(最佳单次/平均)；中国选手维持仅显示 `NR<=200`，其他地区按 `WR > CR > NR` 优先级显示 `<=100` 的排名，排名为 1 时隐藏数字
- **WCA 成绩图**：生成个人最佳记录图片（仿 WCA 官网）
- **PK 对比**：两位选手单次与平均逐项比较，标注优势项 (☆/★)，统计总分 (⭐)
- **one PK 对比**：两位选手在 one 平台逐项比较单次与平均，标注优势项并统计总分
- **双平台自我 PK**：同一选手的 WCA 与 one 成绩逐项对比，保留可映射到 WCA 体系的项目
- **one 查询**：查询 one 平台个人最佳记录
- **跨平台 PR**：合并 WCA 与 one 的最佳成绩，输出双平台合并后的个人 PR
- **跨平台 PRPK**：比较两位选手在 WCA + one 双平台下的综合最佳成绩
- **宿敌查询**：统计在共同项目中"单次与平均均优于你"的选手数量（世界/洲/地区），人数少于等于 10 时会列出名单
- **近期比赛**：获取近期在中国举办的 WCA 比赛列表

## 指令

| 命令 | 描述 | 示例 |
|------|------|------|
| `/cube帮助` | 展示所有命令 | `/cube帮助` |
| `/wca` | 查询 wca 个人记录| `/wca 2026ZHANG01` |
| `/one` | 查询 one 平台个人记录 | `/one 1234` |
| `/wcapic` | 生成个人最佳记录图片 | `/wcapic 张伟` |
| `/wcapk` | 比较两位选手的 wca 成绩 | `/wcapk 张伟 李华` |
| `/onepk` | 比较两位选手的 one 成绩 | `/onepk 张伟 李华` |
| `/pktwo` | 比较同一选手在 WCA 与 one 两个平台的成绩 | `/pktwo 2026ZHANG01 1234` |
| `/pr` | 查询双平台 PR；同名可手动指定 WCAID/oneID | `/pr 2026ZHANG01 1234` |
| `/prpk` | 比较两位选手的双平台 PR 成绩；同名可手动指定双方 WCAID/oneID | `/prpk 李华 张伟` |
| `/wca绑定` | 将当前 QQ 和 WCAID 绑定 | `/wca绑定 2026ZHANG01` |
| `/宿敌` | 查询宿敌 | `/宿敌 2026ZHANG01` |
| `/版本` | 查询宿敌后端当前数据库导出日期 | `/版本` |
| `/近期比赛` | 查询近期在中国举办的 WCA 比赛 | `/近期比赛` |

## 配置

- `enable_command_reaction`：是否启用命令接收后的轻量表情反馈，默认开启
- `command_reaction_emoji_id`：QQ / aiocqhttp 使用的贴表情 `emoji_id`，默认 `124`
- 当前仅 `/wcapk`、`/pr`、`/prpk`、`/宿敌`、`/近期比赛` 会使用这套即时反馈

## 外部依赖

- WCA 官方 API：用于个人成绩查询
- `wca.huizhi.pro`：用于宿敌查询
- `cubing.com` API：用于近期比赛查询
- `ss.sxmfxh.com`：用于 one 平台成绩查询

## 绑定说明

- `/wca绑定 <WCAID或姓名>` 会把当前 QQ 绑定到对应 WCA 选手
- 绑定时如果姓名重名，插件会提示改用唯一的 `WCAID`
- 绑定成功后，直接发送 `/wca` 即可查询自己的成绩
- 发送 `/wca @某人` 时，如果对方已绑定 WCAID，也可以直接查询对方成绩
- 绑定信息保存在插件数据目录下的 `wca_bindings.json`
